# Econ+AI Meeting
## University of Zurich, May 15, 2026

This repository is a compact workshop workspace for a presentation and a live-coded Streamlit demo app.

## Structure

- `docs/references/presentation/econ_ai_workshop.pdf` - presentation deck for the talk.
- `app/app.py` - Streamlit entry point for the live demo.
- `app/data/` - small local datasets for the demo.
- `app/src/` - optional helper code if the demo grows.
- `app/pages/` - optional Streamlit pages.

## Run The Demo App

```bash
cd app
streamlit run app.py
```
