# Scripts

Keep helper scripts grouped by purpose so live-coding support files stay easy to find.

- `chat/` - chat prompts, chat API experiments, and conversation utilities.
- `db/` - data download, database setup, query, and ingestion utilities.
- `plot/` - chart generation, figure export, and plotting experiments.
- `mcp/` - MCP server/client experiments and connector utilities.

Add another clearly named subfolder only when a script does not fit one of these buckets.
